# A.B.E. Public Use Guidelines

A.B.E. is a constitutional audit engine built for public benefit.  
Use is granted under the following conditions:

- No commercial sale or monetization.  
- No paywalls.  
- No integration into proprietary systems.  
- All forks must remain public and free.  
- All versions must credit **Terra Dawn Shouse**.  
- Local-only architecture must be preserved.  
- No user tracking, analytics, cookies, or telemetry.  
- Engine outputs must not be altered to misrepresent results.  

Violations terminate permission under CC BY-NC 4.0.

This engine exists to protect **rights**, not to be exploited by corporations.
