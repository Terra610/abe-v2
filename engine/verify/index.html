Verify Audit Receipt (SHA-256)

Use this tool to confirm that an A.B.E. audit output or receipt has not been altered since the hash was generated. All verification is local-only and never leaves your device.

If you prefer, leave this blank and upload the original file instead.

If a file is selected, it takes priority over the pasted text.

This tool uses the browser's built-in Web Crypto API to compute a SHA-256 hash. Any party (court, counsel, investigator, journalist) can recompute the hash independently to confirm that the audit output has not been modified since the receipt was generated.
